# 🧠 Rick and Morty GraphQL Explorer

## 📘 Overview
This project queries the **Rick and Morty GraphQL API** to display paginated episodes using **Next.js**, **TypeScript**, and **Apollo Client**.

---

## 🧰 Tech Stack
- Next.js  
- TypeScript  
- Tailwind CSS  
- Apollo Client  
- GraphQL
